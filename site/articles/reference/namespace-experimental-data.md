# Namespace: Avalonia.Experimental.Data

This namespace contains the public typed-binding factories used by advanced TreeDataGrid column scenarios.

## Functional Groups

### Binding Factories

- [`TypedBinding<TIn>`](xref:Avalonia.Experimental.Data.TypedBinding`1)
- [`TypedBinding<TIn,TOut>`](xref:Avalonia.Experimental.Data.TypedBinding`2)

## Guidance

Prefer standard column constructors first. Move to typed bindings when you need explicit read/write delegates, fallback behavior, or advanced binding reuse outside standard XAML bindings.
